# Project Organization Plan

To make the project cleaner and more maintainable, we will reorganize the file structure. This involves moving assets into a dedicated `assets/` directory and updating all references.

## 1. Directory Structure Goal

```
portfolio/
├── assets/
│   ├── css/
│   │   └── style.css
│   ├── js/
│   │   └── script.js
│   ├── img/
│   │   ├── satadal.jpg
│   │   └── (other images from images/ folder)
│   └── data/
│       ├── blogs.json
│       └── images.json
├── blog/
│   └── (individual blog posts)
├── scripts/
│   └── (dev/utility scripts)
├── index.html
├── projects.html
├── gallery.html
├── blog.html
├── contact.html
└── gemini.md
```

## 2. Action Items

### Step 1: Create Directories
- Create `assets/css`
- Create `assets/js`
- Create `assets/img`
- Create `assets/data`

### Step 2: Move Files
- Move `style.css` → `assets/css/style.css`
- Move `script.js` → `assets/js/script.js`
- Move `satadal.jpg` → `assets/img/satadal.jpg`
- Move contents of `images/` → `assets/img/`
- Move `blogs.json` → `assets/data/blogs.json`
- Move `images.json` → `assets/data/images.json`
- Delete empty `images/` directory.

### Step 3: Update References (Crucial)

**HTML Files (`index.html`, `projects.html`, etc.):**
- Update CSS link: `href="style.css"` → `href="assets/css/style.css"`
- Update JS script: `src="script.js"` → `src="assets/js/script.js"`
- Update Profile Image: `src="satadal.jpg"` → `src="assets/img/satadal.jpg"`
- Update Favicon (if any).

**CSS File (`assets/css/style.css`):**
- Check for any `url('images/...')` references and update to `url('../img/...')`.

**JS File (`assets/js/script.js`):**
- Update fetch calls:
  - `fetch('blogs.json')` → `fetch('assets/data/blogs.json')`
  - `fetch('images.json')` → `fetch('assets/data/images.json')`
- Update image paths generated by JS:
  - `img.src = 'images/' + ...` → `img.src = 'assets/img/' + ...`

## 3. Verification
- Check Home page load (styles, scripts, profile image).
- Check Gallery page (loading images from JSON).
- Check Blog page (loading posts from JSON).
- Check Console for 404 errors.
